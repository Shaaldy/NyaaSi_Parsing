Данный python-скрипт позволяет пропарсить сайт Nyaa.Si. Кокретно запишет за последнее время большинство аниме с сайта, с их подробностью и ссылкой на скачивание. Более того позволяет выбранное аниме (можно несколько) скачивать во время скрипта в заранее написанный путь.

Для его использования необходимо
1) установить прокси сервер или VPN - соединение
2) изменить хэдер на свой (подробности в коде (см. комментарии))
3) проверить что файлы находятся в одной директории
4) для изменения deltaDate (за какой промежуток времени вы хотите отсмотреть) измените в коде. По умолчанию 3
4) запустить NyaaSiWriter.py 
5) В консоли ответить Y/N чтобы скачать файлы
